# Тестирование API сервиса доставки скутеров

[URL сервиса](https://qa-scooter.praktikum-services.ru/)

[Документация API](qa-scooter.praktikum-services.ru/docs/)

## Подготовка проекта
1. Создать Maven‑проект
2. Подключить следующие библиотеки:
   - JUnit 4 — для написания и запуска тестов
   - RestAssured — для тестирования API
   - Allure — для создания отчётов по результатам тестирования

## Задание
### 1. Создание курьера
**Проверяемые сценарии:**
- Успешное создание курьера
- Попытка создать двух одинаковых курьеров: ожидается ошибка
- Проверка обязательных полей: запрос без всех обязательных полей должен возвращать ошибку
- Запрос возвращает правильный код ответа
- В случае успешного запроса в ответе должно быть `ok: true`
- При отсутствии одного из обязательных полей возвращается ошибка
- При попытке создать курьера с уже существующим логином возвращается ошибка

### 2. Логин курьера
**Проверяемые сценарии:**
- Успешная авторизация курьера
- Проверка обязательных полей для авторизации: запрос без всех обязательных полей должен возвращать ошибку
- Некорректный логин или пароль: ожидается ошибка
- Отсутствие какого‑либо поля в запросе авторизации приводит к ошибке
- Попытка авторизоваться под несуществующим пользователем: ожидается ошибка
- При успешной авторизации в ответе должен возвращаться `id` курьера

### 3. Создание заказа
**Проверяемые сценарии:**
- Возможность указать цвет `BLACK`
- Возможность указать цвет `GREY`
- Возможность указать оба цвета (`BLACK` и `GREY`)
- Возможность не указывать цвет вовсе
- В теле ответа должен присутствовать параметр `track`

### 4. Список заказов
**Проверяемый сценарий:**
- При запросе списка заказов в теле ответа возвращается массив (список) заказов

## Отчётность
По результатам выполнения тестов необходимо сформировать отчёт Allure

## Требования к реализации тестов
* Независимость тестов. Каждый тест должен выполняться независимо от других
* Очистка данных. После выполнения теста все созданные сущности должны быть удалены

## Команды
Эта команда для очищения проекта и запуска тестов с нуля

`mvn clean test`

Эта команда для запуска локального веб‑сервера для просмотра отчёта Allure

`mvn allure:serve`
